## 한계
- 시간 부족으로 전체적인 디자인 수정과, 메모할 수 있는 화면까지 구현하는 데는 실패

## 소감
- 앱 개발은 처음이라 막막했는데 인터넷과 관련 서적의 도움을 받으니 조금 할 만했다. 과제와 개인 생활로 시간이 빠듯해서 완성을 하진 못했지만 개인적인 코딩 실력 향상에 많은 도움이 되었다.
